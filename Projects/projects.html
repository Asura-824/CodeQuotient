My projects

As a Data Analyst and Python Developer, I present a series of hands-on projects that have helped me level up my skills and sharpen my critical thinking. While working on these projects, I applied various data analysis and visualization techniques to derive insights and communicate findings effectively. My Python development work includes projects such as QR Code Generator and Phone Data Masking, along with other tools and analyses that demonstrate practical problem-solving and technical fluency

Rock paper scissor

This project is a simple implementation of the classic Rock-Paper-Scissors game using Python. It allows users to play against the computer, keeping track of scores and determining the winner based on standard game rules.

Number Guessing

This project is a simple number guessing game implemented in Python. The program randomly selects a number within a specified range, and the player must guess the number with feedback provided for each guess.

Face Detection

This project involves implementing face detection using Python and the OpenCV library. The program captures video from a webcam and uses pre-trained Haar Cascade classifiers to detect and highlight faces in real-time.

Stopwatch & Timer

This project implements a stopwatch and countdown timer using Python. The stopwatch allows users to start, stop, and reset the timer, while the countdown timer lets users set a specific time duration and counts down to zero.

QR Code Generator

This project involves creating a QR code using Python. It allows users to input text or URLs and generates a corresponding QR code image. The project utilizes libraries such as qrcode and Pillow for image processing.

Phone Data Masking

This project focuses on masking sensitive information, specifically phone numbers, in a dataset. It involves using regular expressions and string manipulation techniques in Python to anonymize personal data while retaining its format.

EV Population

This project focuses on analyzing electric vehicle (EV) data in Washington State (WA) to understand trends, adoption rates, and other relevant metrics related to EV usage and infrastructure.

Indian Census

In this project, I have analyzed the Indian Census data from 2011 to extract meaningful insights about the population distribution, literacy rates, and other demographic factors across different states and union territories in India.

Netflix

This project focuses on analyzing Netflix user data to understand various aspects of customer behavior, subscription patterns, and engagement. It involves loading the dataset, performing initial data inspection, and setting the groundwork for deeper analytical insights.

Covid-19

This project exploratory do the data analysis of COVID-19 statistics using publicly available datasets. The analysis involves loading, cleaning, and summarizing key metrics related to the pandemic.

Spotify

This project is a data analysis of a Spotify dataset, exploring user listening habits and track information. The analysis is performed in a Jupyter notebook (Spotify.ipynb) using Python and various data science libraries.
